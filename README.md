First steps to DDPC

Download
```bash
cd %HOME/Downloads
curl -fsSL -o ddpc  "https://raw.github.com/ByteCraftServices/ddpc/main/get/download.sh"

```
Download & starten
```bash
cd %HOME/Downloads
curl -fsSL -o ddpc  "https://raw.github.com/ByteCraftServices/ddpc/main/get/download.sh" && chmod +x ./ddpc && ./ddpc

```
Alpha
